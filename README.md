# adbrixrm_flutter

AdBrixRM Plugin for Flutter Project

## Support Platform

- Android 4.1 +
- iOS 10 +

## Current AdBrixRM SDK version

- Android :  1.1.2210
- iOS : 1.6.5230

## Install the Plugin

Add dependencies on pubspec.yaml

```
dependencies:
  adbrixrm_flutter: ^1.0.2
```
